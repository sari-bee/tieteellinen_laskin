## Asennus ja käynnistys

Asenna riippuvuudet komennolla

```bash
poetry install
```

Käynnistä sovellus komennolla

```bash
poetry run python3 src/index.py
```

Suorita testit komennolla

```bash
poetry run pytest src
```
